## 🎮 Ping Pong Game Features

- Built 100% in **vanilla JavaScript**
- 🖱 Mouse & Touch support
- 📈 Speed increases as you score
- 🎨 Dynamic background changes
- 💾 High Score saved via `localStorage`
- 💀 Game Over screen with replay option
